# bb8-soul

Temporary home for the robot's constitution, hierarchy, and personality files.

This directory is intentionally plain Git for now. Later it should become a Git submodule backed by a separate repository.

## Rules for now

- Humans edit these files directly.
- The agent may read these files as context.
- The agent should not commit directly here.
- Future agent-authored proposals should go under `proposed_changes/` and be reviewed by a human.
